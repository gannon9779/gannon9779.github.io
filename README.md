# Nittany STEM Tutoring — GitHub Pages

Static multi-page site for class / portfolio.

## Pages

| File | Purpose |
|------|---------|
| `index.html` | Homepage — logo, consultancy description, cards linking everywhere |
| `features.html` | Deeper “why us” + process |
| `about.html` | Mission copy + **~1 min pitch video** |
| `services.html` | Service cards + pricing |
| `contact.html` | **Contact form** (FormSubmit) + **Calendly** embed |
| `newsletter.html` | Newsletter sample: offer copy, branding, site link, founder article |

Shared styles: `assets/styles.css`.

## Pitch video (`about.html`)

The About page plays **`assets/about-pitch.mp4`** (~67 MB, under GitHub’s 100 MB limit). Replace that file if you re-record; keep replacements under **100 MB** or use a YouTube `<iframe>` embed instead.

## Contact form

The form posts to [FormSubmit](https://formsubmit.co/) at `hello@nittanystemtutoring.com`. The first time a new address receives a submission, FormSubmit sends a confirmation email—click it so future messages arrive normally.
